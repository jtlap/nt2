Supported SIMD instruction sets

At the moment the library supports on x86 platforms the following instruction sets :

  • SSE2
  • SSE3
  • SSSE3
  • SSE4_1
  • SSE4_2
  • AVX
  • XOP

The other platform supported is Powerpc with the Altivec instruction set.
